# User Story Template

- Title: `As a <persona>, I want <capability>, so that <outcome>`
- Description:
  - context
  - business logic
  - constraints
- Scenario:
  - happy path
  - alternative flow
  - edge case
  - failure path
  - expected outcome
- Acceptance criteria:
  - Given / When / Then
  - or checklist form
- Estimate:
  - use the team's agreed scale
